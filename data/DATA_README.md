# Data availability note (placeholder)

The raw item-level dataset used in this repository cannot be publicly shared due to privacy issues.

To reproduce the analyses, the following files are expected in `data/`:
- compost_items.csv
  - must include the item columns listed in `R/00_config.R` (vars_full / vars_pruned)
  - ordinal-coded responses (numeric levels)
- cfa_model_syntax_compost.xlsx
  - must include columns: model, syntax
  - sheet name is defined in `R/00_config.R`


De-identified data are available from the corresponding author upon reasonable request and subject to ethical restrictions to protect participant privacy (including potentially identifying location information).

If you wish to request access to the data, please contact: takeshiabe@tottori-u.ac.jp.
